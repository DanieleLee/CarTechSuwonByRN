
## 1 About
자동차협회 관리자는 협력업체를 등록/수정/삭제 할 수 있다. 협력 업체는 적립된 포인트로 협회몰에서 필요 물품을 구매할 수 있다. <br />
 💡 현재는 로그인 -> 협회몰 구매가능 물품리스트 조회만 가능(무한스크롤)
